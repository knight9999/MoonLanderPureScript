# ビルドの仕方

```
$ npm install
$ npx spago install
$ npm run build
```

# ゲームの開き方

`public/index.html`をブラウザ(Chrome, Firefoxなど)で開きます。
（現在のところPCだけです。スマホは未対応）

# ローカルサーバーを起動する

```
$ npm run start
```

で3000ポートで立ち上がります。この場合、`http://localhost:3000`にアクセスすれば確認出来ます。
(スマートフォンからの場合、`http://PCのIPアドレス:3000` でアクセスする)
